# Nyxie Wonderland

夜希（Nyxie）的交互式角色作品集网页。首页使用连续角色动画、仙境视觉和滚动章节构成沉浸式展示。

## 本地运行

项目不需要安装依赖，使用 Node.js 启动静态服务器：

```powershell
npm run dev
```

然后打开 <http://127.0.0.1:4173>。

## 项目结构

- `index.html`：页面内容与结构
- `styles.css`：视觉、动效与响应式布局
- `app.js`：角色动画、页面交互与章节控制
- `assets/media/`：网页使用的图片和视频素材
- `server.mjs`：本地静态服务器

原始生成素材和参考视频仅保留在本地工作目录，不提交到公开仓库。
